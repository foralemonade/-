## 1. 游戏概述

### 1.1 基本信息

| 项目       | 内容                |
| -------- | ----------------- |
| **游戏名称** | 梦游症（Somnambulism） |
| **类型**   | 2D 塔防 + 养成 RPG    |
| **目标平台** | 手机（iOS / Android） |
| **玩家模式** | 单人为主，含多人可选内容      |
| **项目性质** | indie 项目          |

### 1.2 核心卖点

- **末世 × 卡通反差美学**：末日废土的沉重背景，配上动漫卡通的温暖人物画风，形成强烈视觉张力与情感反差；
- **移动城堡 + 派系搭配**：玩家驾驶一座可扩展的移动城堡，在城堡上自由混载不同派系的生物，激活派系组合效果；
- **六大陆探索**：五大派系各占一片大陆，加上中立混战大陆，地图探索与派系交互构成开放世界感；
- **深度策略**：塔防布置、生物派系搭配、建筑功能模块选择，三层策略叠加，上手简单、精通难；
- **无限挑战**：定时刷新的 Roguelike 浪潮模式，提供长线游玩动力。

### 1.3 一句话介绍

> 在末日降临的世界里，一个被卡通画风包裹的孤独旅者，驾驶着自己的移动城堡，穿越由五大派系割据的六片大陆，以独特的"生物调度"战术，在不断涌来的威胁中寻找生存真相。

## 2. 世界观设定

### 2.1 背景故事

世界进入了被称为 **"末鸣纪"** 的时代。

大灾变之后，文明的骨架已经崩塌，人类没有被外星入侵者消灭，也没有被病毒击倒——他们被自己打垮了。

派系之争是末世的本质：资源、信仰、恐惧、傲慢，在旧文明碎片上孕育出了五支截然不同的势力。他们各自占据一片大陆，彼此敌视，偶尔交战，并在中间那片无名混战大陆里互相渗透与倾轧。

玩家扮演一名末世界原住民——**"梦游者"**。

**梦游者**不属于任何派系，却又游走于所有派系之间。他们的名字来源于一个末世传说：*"在世界崩塌时入睡，在世界重建时行走"*——那些在末日降临之夜没有醒来、却以某种方式活下来的人，被称为梦游者。

梦游者不是英雄，也不是救世主。他们只是……活下来了。

### 2.2 世界地图结构

<img title="" src="file:///D:/%E6%A1%8C%E9%9D%A2/work1-1/%E6%A2%A6%E6%B8%B8%E7%97%87/%E6%A2%A6%E6%B8%B8%E7%97%87%E5%85%AD%E5%A4%A7%E9%99%86%E5%B1%95%E7%A4%BA%E5%9B%BE.png" alt="" style="zoom:25%;" data-align="left">

- **五片派系大陆**：每片大陆由一个派系主导，拥有独特地形、生物、建筑配件；
- **中心混战大陆**：中立地带，五大派系均在此留有势力，资源丰富但危险极高，是玩家收集稀有生物和配件的核心场所；
- **移动城堡**是玩家穿越各大陆的交通工具与战斗核心。

### 2.3 世界规则

- 派系之间没有法律，只有力量与利益；
- 梦游者是例外：因为"不属于任何人"，各派系都愿意与梦游者交易，但也都不真正信任他；
- 生物并非全是怪物——许多生物只是乱世中失去归属的存在，与梦游者的关系更像是"雇佣"和"共生"。

## 3. 五大派系与大陆

> 以下为派系框架设计，具体细节（名称、生物外形、建筑风格）在开发阶段可继续填充。

### 派系设计原则

每个派系须具备：

- 独特的**意识形态**（他们为何而战？）
- 标志性的**生物类型**（战斗风格倾向）
- 独特的**城堡配件**（影响城堡功能）
- 专属**大陆地貌**（影响关卡设计）
- 与其他派系的**关系**（盟友/中立/宿敌）

### 3.1 派系框架（占位设计）

| 派系编号 | 核心理念方向 | 生物风格方向     | 建筑/配件风格  | 大陆地貌方向     | 地区特色                   |
| ---- | ------ | ---------- | -------- | ---------- | ---------------------- |
| 派系 Ⅰ | 秩序·技术  | 机械改造生物     | 精密工业模块   | 废弃城市·钢铁废土  | 怪物很多，血量很高（基础质量高），但是上限低 |
| 派系 Ⅱ | 信仰·神秘  | 灵体·祭祀生物    | 神殿式祭坛组件  | 焦土高原·神圣遗迹  | 唯一神教，宗族信仰              |
| 派系 Ⅲ | 自然·野性  | 野兽·变异植物    | 藤蔓·骨制配件  | 异变森林·毒雾沼泽  | 有毒，回复能力强               |
| 派系 Ⅳ | 商业·混沌  | 雇佣兵·变种人    | 拼凑式市场模块  | 流沙大陆·地下贸易城 | 个体质量高，关系错综复杂。不成帮派      |
| 派系 Ⅴ | 记忆·复古  | 异变动物·旧世界生灵 | 旧世纪博物馆配件 | 冰封大陆·旧文明残骸 | 移速慢各元素抗性高（需真实伤害？）      |

### 3.2 中心混战大陆

- **名称**：灰心岛 
- **特征**：所有派系生物均可能出现；稀有生物和顶级配件集中产出；关卡难度最高；定期发生"派系事件"影响地图格局；
- **功能**：玩家中后期核心探索区域。

## 4. 核心玩法设计

### 4.1 玩法结构总览

```
【大地图层】
  ├── 选择大陆 → 进入该派系大陆
  ├── 大陆内探索（关卡入口、商人、配件、生物捕获）
  └── 前往混战大陆

【战斗层（塔防）】
  ├── 进入关卡
  ├── 布置城堡位置 + 安装生物
  ├── 波次开始，生物自动防御
  ├── 主角支援技能辅助
  └── 通关/失败 → 结算奖励

【养成层】
  ├── 主角成长（等级、技能）
  ├── 城堡模块扩展
  └── 生物收集与培养
```

### 4.2 战斗基础流程

1. **进入关卡前**：查看敌方波次预览（派系、数量、特殊敌人）；
2. **布置阶段**：将城堡停在地图某位置；将收集到的生物安置到城堡对应槽位上；
3. **战斗阶段**：
   - 敌人沿路线进攻，生物自动开火；
   - 主角可在城堡上发动支援技能（减速、范围强化、护盾等）；
   - 派系生物组合触发被动效果；
4. **结算阶段**：根据通关评级获得资源、配件、生物碎片等奖励。

### 4.3 敌人设计

敌人来自**人类派系之争**——不是怪物，是人。

- 每个派系的士兵/生物有独特的行进方式和攻击模式；
- 同一波次中可能混入多个派系的敌人；
- 高级关卡会出现**派系精英单位**和**派系 BOSS**；
- 中心混战大陆的敌人是**混编部队**，更难预判。

## 5. 城堡系统

### 5.1 城堡概念

移动城堡是玩家的**核心战斗单位**，也是玩家的"家"。

它不是固定防御工事，而是**可移动、可配置、可扩展**的战斗平台。
城堡在大地图上缓缓移动，进入战场后停驻，上方的生物开始作战。

### 5.2 城堡模块系统

城堡由多个**功能模块**组成，模块可以从各大陆收集配件后解锁或升级：

| 模块类型       | 功能描述              | 解锁来源   |
| ---------- | ----------------- | ------ |
| **生物槽位模块** | 决定城堡可搭载的生物数量与排列方式 | 主线推进   |
| **防御模块**   | 城堡自身的护甲/护盾值       | 各大陆配件  |
| **能量模块**   | 为主角技能提供能量恢复速度     | 各大陆配件  |
| **派系共鸣模块** | 装备后增强特定派系生物效果     | 特定派系大陆 |
| **侦察模块**   | 可提前查看敌人波次情报       | 混战大陆稀有 |
| **资源模块**   | 战斗中额外产出资源         | 商人购买   |

> **设计原则**：城堡模块的选择本身就是一层策略——玩家需要根据当前大陆/关卡的派系特点来选择最优配置。

### 5.3 城堡生物槽位

- 城堡外观类似一座小型堡垒/要塞，顶部和侧面有若干**生物安置格**；
- 不同位置的安置格可能有位置加成（如高台格射程+1，侧翼格攻击角度扩展等）；
- 生物可混载不同派系，不强制要求同派系。

## 6. 生物系统

### 6.1 生物基础属性

每只生物都有以下基础属性：

- **攻击力**：对敌人造成的伤害
- **攻击速度**：出手频率
- **攻击范围**：覆盖范围（近战/中程/远程/全图）
- **目标优先级**：优先打最近/血量最低/精英敌人等
- **派系标签**：所属哪个派系（可混搭）
- **特殊技能**：每只生物有一个被动或主动特殊能力

### 6.2 派系搭配效果

当城堡上同时放置**同一派系的多只生物**时，激活**派系协同效果**：

| 激活条件     | 效果示例            |
| -------- | --------------- |
| 同派系 2 只  | 攻击力 +15%        |
| 同派系 3 只  | 额外触发派系专属被动      |
| 同派系 4 只+ | 激活派系终极共鸣，全场持续效果 |

当城堡上放置**跨派系**生物时，也可能触发**跨派系反应**（类似原神元素反应）：

> 例：派系Ⅲ（自然）× 派系Ⅰ（技术）→ 触发"生化异变"效果，攻击附带持续伤害  
> 例：派系Ⅱ（信仰）× 派系Ⅴ（记忆）→ 触发"回响"效果，死亡生物短暂复活一次

*（具体反应表在细化阶段设计）*

### 6.3 生物获取方式

- 在各大陆关卡中，通关后有概率**捕获**当地生物；
- 大地图上的**商人**可以购买；
- **混战大陆**掉落稀有跨派系生物；
- 部分生物需要**派系好感度**达标才能收集。

### 6.4 生物培养

- 生物可通过战斗**积累经验**，提升基础属性；
- 收集相同生物碎片可以**觉醒**，解锁进阶外形和新技能；
- 不做养成也能玩，但深度养成带来明显战力提升（非强制肝）。

## 7. 主角与养成系统

### 7.1 主角设定

- **称号**：梦游者
- **性别**：玩家自定义（不影响剧情，仅外形差异）
- **身份**：末世界原住民，不属于任何派系
- **战斗角色**：支援型——不直接参与主线输出，但能影响全局节奏

### 7.2 主角技能体系

主角的技能围绕**"支援"**设计，分为以下流派方向：

| 技能流派    | 方向描述     | 示例技能            |
| ------- | -------- | --------------- |
| **强化流** | 提升生物攻防   | 全场攻击力+20%（持续5秒） |
| **控制流** | 干扰敌人行进   | 全场敌人减速 / 范围眩晕   |
| **治愈流** | 修复城堡或生物  | 恢复城堡护盾 / 复活一只生物 |
| **召唤流** | 临时召唤辅助单位 | 召唤临时炮台 / 召唤派系援军 |

> 玩家可在主角成长过程中**选择技能方向**，形成专属的辅助打法风格。

### 7.3 主角成长路径

```
等级提升
  ├── 解锁/升级技能
  ├── 提升能量上限（可使用技能次数增加）
  └── 解锁新城堡模块配方

探索成就
  ├── 与派系交互 → 提升派系好感度 → 解锁该派系稀有生物
  └── 收集图鉴 → 触发世界观碎片解锁（背景故事片段）
```

### 7.4 派系好感度系统

- 梦游者与每个派系有独立的**好感度值**；
- 好感度通过：在该派系大陆完成关卡、与该派系商人交易、选择支持该派系的地图事件 来提升；
- **好感度阶段解锁内容**：

| 好感阶段 | 解锁内容                |
| ---- | ------------------- |
| 初识   | 可购买该派系基础生物          |
| 友好   | 解锁稀有生物、专属配件         |
| 信赖   | 可进入该派系核心区域、解锁派系传说生物 |
| 至交   | 城堡获得该派系外观主题（限定装饰）   |

## 8. 地图与关卡设计

### 8.1 大地图层

- 整体为六片大陆构成的世界地图，城堡在大陆间移动；
- 每片大陆内部有若干**节点**，节点类型包括：

| 节点类型    | 描述                 |
| ------- | ------------------ |
| ⚔️ 战斗关卡 | 标准塔防战斗             |
| 🏪 派系商人 | 购买生物、配件、道具         |
| 🔍 遗迹探索 | 找到背景故事碎片、稀有配件      |
| ⚡ 随机事件  | 选择题型，影响好感度/获得/失去资源 |
| 🏰 派系据点 | 高难度战斗，通关奖励丰厚       |
| 💀 精英挑战 | 特殊规则的限制战斗          |

### 8.2 关卡内地图设计（2D 横版）

- 关卡为**2D 横版视角**，敌人从一端向城堡行进；
- 城堡固定在画面某一处（自行选择），生物在城堡上自动攻击；
- 有不同的类似地图上机关或者自己造地形以更好的解决怪物。
- 地形可设计为：
  - **单路直线**（基础关卡）
  - **高低地形**（影响射程和遮蔽）
  - **多路分流**（考验生物覆盖范围）
  - **移动平台**（关卡特殊机制）

### 8.3 波次设计

- 每关分为 **3-6 波**敌人；
- 波次间有短暂喘息时间（可快进or跳过）；
- 最后一波必有**精英/小 BOSS**出现；
- 大陆 BOSS 关为单独特殊战斗（BOSS 专属行为逻辑）。

## 9. 无限挑战模式（排位吗，待考虑）

### 9.1 模式概念

无限挑战模式是独立于大地图之外的**重复可玩内容**，定时（如每周/每日）刷新。

- 类 Roguelike 结构：每次进入随机选择路线、随机生物组合、随机关卡参数；
- 不消耗大地图进度，独立结算；
- 成绩与奖励榜单（可选：本地记录 or 加入排行）。

### 9.2 进行流程

```
进入挑战 → 选择初始派系偏向 → 
第1关（简单） → 选择强化卡牌 → 
第2关（中等） → 选择强化卡牌 → 
... 
第N关（极难） → 结算得分 → 兑换奖励
```

### 9.3 强化卡牌（Roguelike 随机加成）

每通过一关后，从三张随机强化卡中选一张，效果在本次挑战内生效：

> 示例卡牌：
> 
> - "派系共鸣增幅"：本局派系协同伤害+30%
> - "城堡铁壁"：城堡护盾值+50%
> - "生物狂暴"：所有生物攻速+25%，但防御-15%
> - "召唤援军"：每波波次结束后召唤随机临时生物一只

### 9.4 刷新机制

- **每日刷新**：小型挑战，奖励日常资源；
- **每周刷新**：大型挑战，奖励稀有配件/生物碎片；
- 每次刷新更换地图参数和敌方派系组合，防止固化打法。

## 10. 美术风格

### 10.1 风格定位

**"末世卡通反差风"**

- **世界观层面**：末日废土、天灾与人祸、断壁残垣、人类内战——沉重、荒芜、充满绝望感；
- **画面层面**：明亮的卡通线条、饱和度较高的色彩、Q 版/动漫比例的角色——可爱、温暖、充满生命力；
- **张力来源**：内容的残酷性和视觉的温柔感形成反差，这种"反差萌"是本作视觉辨识度的核心。

### 10.2 参考方向

| 风格参考    | 借鉴方向             |
| ------- | ---------------- |
| 《明日方舟》  | 末世世界观 + 动漫插画风格   |
| 《王国保卫战》 | 2D 塔防的视觉节奏和单位可爱化 |
| 《辐射》系列  | 末日废土的文化残骸感       |
| 《咩咩启示录》 | 卡通化处理黑暗主题的方式     |

### 10.3 角色设计原则

- 生物造型：不同派系有统一的设计语言（如派系Ⅰ生物有机械感，派系Ⅲ生物有植物纹理）；
- 主角（梦游者）：中立感强，外形不归属于任何派系，有独特的"睡眠/梦境"视觉符号；
- 城堡：整体可爱，但细节展示历史感（修补痕迹、旧世界零件拼凑）。

### 10.4 UI 风格

- 手机竖屏/横屏均可支持（优先横屏战斗视角）；
- UI 风格偏向手绘板书/羊皮纸质感，与世界观相融；
- 信息密度适中——手机屏幕不宜过于拥挤。
